Jason의 외국어 Study 사용법 안내

[ 기본 개념 ]

• 기본 학습 개념은 외국어 짧은 문장을 자주 들으면서 따라 읽기를 하면서 외우는 것이다.

• 이를 위해서 암기를 필요로 하는 문장에 대해서 20 ~ 500 문장으로 되어 있다.

• 문장은 수준에 따라 다른데 가능하면 10단어 이내의 짧은 문장으로 한다.

• 문장은 모두 mp3 파일로 저장이 되어 있다.

• 그룹은 전체 암기 대상수가 많을 경우에 적정 수준으로 나누어서 공부하도록 한다.

• 암기완료를 빠르게 진행하고, 다시 처음부터 반복한다.


[ 어플 사용 방법 ]

• 스터디용 웹 url 정보를 스마트폰 바탕화면 바로가기로 만들어 둔다.

• 최초 스터디 url을 접속한 후에, 초기화 버튼을 누른다.

• 암기 대상을 적당한 개수로 나누어서 공부하도록 그룹을 나눈다다.

• 화면 하단에 암기해야할 총단어수가 표시되며, 위에서 정한 그룹 단위에 따라 나뉘어진 그룹의 개수를 확인할 수 있다.

• 불러오기는 다시 웹에 접속했을때 기존에 공부하던 내용을 불러온다.

• Group No 는 현재 암기하고 있는 그룹을 이야기 한다.

• 이전그룹, 다음그룹 버튼은 그룹을 이동하면서 암기할 때 사용한다.

• 그룹조정은 암기완료된 항목을 제외하고 현재 남은 대상을 다시 재그룹할 때 사용한다.

• 이전, 다음 버튼은 현재 그룹내에서 암기 대상을 이동할 때 사용한다.

• 자막은 현재 대상의 한글/외국어 자막을 확일할 때 사용한다.

• "PLAY"는 현재 암기 대상의 mp3 파일을 1회 재생할 때 사용한다.

• "PLAY R"는 현재 암기 대상의 mp3 파일을 무한 재생할 때 사용한다.

• "PLAY A"는 현재그룹내 파일을 순차적으로 무한 재생할때 사용하며, 토글 버튼으로 다시 누르면 종료된다.

• 암기완료는 현재 학습 대상을 암기완료로 처리하고, 리스트에서 삭제한 후 총 암기완료 숫자를 하나 늘린다.

• 하단에는 날짜별로 암기 완료한 개수가 표시 된다.

[ 가운데 노란 터치 영역 활용 ]

• 노란 영역은 스마트폰 사용시 한손으로 조작 가능하게 터치 기능이 들어가 있다.

• 한번 터치하면 현재 음원을 play 하고, 좌/우 슬라이딩을 하면 이전/다음 대상으로 변경이 된다. 하단으로 내리면 자막보기가 된다.

• 더블클릭하면 암기 완료 처리가 되고, 자동으로 다음 대상으로 이동하여 play가 된다.
